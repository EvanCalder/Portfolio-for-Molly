# Molly Youngblood — Portfolio

Immersive WebGL portfolio for **Molly Youngblood**, Senior AI/ML Engineer. Includes an embedded JotForm conversational agent.

## Run (Windows)

**Easiest:** double-click **`start.bat`** in this folder.

Or in a terminal:

```bash
npm install
npm run dev
```

Open **http://127.0.0.1:5138/**

## Personalize

Edit `scripts/resume-profile.json`, then:

```bash
npm run sync-assets
npm run personalize
```
